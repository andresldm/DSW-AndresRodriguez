OVI en formato WEB con HTML5 y CSS3

Objetivo General

    informar sobre los componentes que usa una pagina web mediante el uso de las herramientas de Html5 y de Css3

Objetivos especificos

    diseño de una presentacion Ovi

    dividir o seccionar el contenido

    uso de contenido multimedia

Descripcion Html5

html5 es una combinacion de nuevas etiquetas de Html, propiedades css3, javascript y otras tecnologias de apoyo, nuevas sintaxis para crar paginas web co las etiquetas antiguas, estos avances son consecuencia del avance tecnologico y las necesidades de los usuarios por las limitaciones que este tenia como la parte multimedia que dependia de los plugins.

Celaya Luna, A. (2014). Creación de páginas web: HTML 5. Málaga, Spain: Editorial ICB. Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045?page=28.

Descripcion Css3

css3 es el marco que permite la transfornacion de los elementos dentro de Html5, como lo son rotar, escalar, animar etc. estos efectos visuales permiten dar mejores apariencias como el acceso a nuevas fuentes
